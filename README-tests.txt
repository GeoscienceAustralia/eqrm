Unit Tests
==========

To run the unit tests, from eqrm_core do;
   python test_all.py
   
   
Implementation Tests
====================

To run the scenarios and check that the scenario results have not
changed, from eqrm_core do;  
   python check_scenarios.py
   
   
Plotting unit Tests
===================

To run the plotting tests, firstly go to;
   eqrm_code/plotting
Then do;
   python test_all.py

Plotting Demos
==============
To run all of the plotting api's go to;
   demo/plot
Then do;
   python execute_all_demos.py
